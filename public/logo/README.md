# Assets de logo — Tandera Tech

Imagens de **referência** extraídas do brandbook 2026. Servem para consulta
visual e como placeholder durante o desenvolvimento.

| Arquivo                          | O que é                                              | Fundo        |
|----------------------------------|------------------------------------------------------|--------------|
| `tandera-lockup-dark.png`        | Lockup horizontal (símbolo + "tandera.tech")         | escuro       |
| `tandera-lockup-light.png`       | Lockup horizontal, símbolo/letra em preto            | claro/branco |
| `tandera-icon-circle.png`        | Símbolo isolado no círculo, gradiente-assinatura     | escuro       |

## Importante

- O **arquivo-mestre vetorial** é `tandera-brandbook.ai`, na pasta `LOGO` da
  empresa. Para produção, exporte SVGs limpos a partir dele (com fundo
  transparente) e coloque-os em `public/logo/` do projeto Next.js.
- Os exemplos de código nas references usam caminhos como
  `/logo/tandera-lockup-light.svg` — substitua pelos SVGs exportados, ou ajuste
  para `.png` apontando para estes arquivos enquanto os vetores não existirem.
- Não recolora, distorça ou gire o símbolo fora das versões do brandbook
  (gradiente aqua, branco, preto). Veja as regras de uso no `SKILL.md`.
